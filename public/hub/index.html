Rapport för Tvåkanten genererad och deployad.

Vad som återstår innan lansering: Fas 3 (timeout-fix — enhanced scan tar ~90s, Cloudflare kopplar ner vid 100s) är teknisk blocker. Fas 2 (lead capture/e-post) blockerar affärsmodellen. Fas 4 (Stripe) sist.

Nästa steg: Fas 3 (polling-arkitektur) → Fas 2 (e-postgate) → lansering → Fas 4 (Stripe).

4 faser för att gå från fungerande endpoint till betalande kunder

Fas 1 ✓ Klar 2026-04-27

Koppla ihop endpointen

Enhanced-scan är nu primärt flöde. UI:t kör analyze.pipod.net med stadsval direkt mot /api/enhanced-scan.

  • ✓EnhancedReport.tsx — status badges + synthesis markdown renderer
  • ✓useAnalysis — analyze(url, city) → enhancedReport state
  • ✓Stadsval-fält i UI — city → city-guard i API
  • ✓Tavily katalogkoll + NAP + 2×5 recensioner + sampleNote
✓ Live på analyze.pipod.net
Fas 2 1 dag

Lead capture

Samla e-poster innan rapporten visas. Noll leads idag — detta är det blockerande affärsproblemet.

  • →Byt "Kör Full Analys"-knappen mot e-postformulär
  • →Spara mail + URL i leads-fil/DB
  • →Kör enhanced scan → visa rapport direkt
  • →E-posta även rapporten till användaren via SMTP
⚠ Blockerar: Fas 1 måste vara klar
Fas 3 1 dag

Timeout-fix

Enhanced scan tar ~2 min. Cloudflare kopplar ner efter 100s. Utan detta crashar alla anrop via analyze.pipod.net.

  • →POST /api/enhanced-scan/start — startar scan, returnerar job_id
  • →GET /api/enhanced-scan/status/[id] — polling var 5s
  • →Progress-text per steg: "Analyserar signaler… Skapar plan…"
  • →Spara jobb i Redis (finns) eller in-memory Map
🔴 Teknisk blocker för produktion
Fas 4 1–2 dagar

Stripe — ta betalt

Byt e-postformuläret mot betalning. Fas 2 och 3 måste vara klara.

  • →Stripe Checkout (engångspris, t.ex. 299 kr)
  • →Webhook: verifiera betalning → starta scan
  • →Betalnings-token i session → låser upp rapporten
  • →Skicka rapport som PDF/HTML till köparens mail
Kräver: Fas 1 + 2 + 3
Snabbaste vägen till första betalande kund
Fas 1 → Fas 3 → Fas 2 (e-post) → lansera → validera intresse → Fas 4 (Stripe)
