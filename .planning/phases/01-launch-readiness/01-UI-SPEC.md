## Typography

**Font assignment rule:** `h1, h2, h3, h4 { font-family: var(--f-display) }` and `em { font-family: var(--f-serif); font-style: italic }` ‚Äî apply globally in `globals.css`.

| Role | Size | Weight | Line Height | Letter Spacing | Font | Maps to |
|------|------|--------|-------------|---------------|------|---------|
| Display | clamp(48px, 9vw, 100px) | 700 / 800 | 0.95‚Äì1.05 | ‚àí0.03 to ‚àí0.04em | Syne | Hero h1 (800), section h2 (800), stat numbers (800), reason ordinals (800), hero subheading via Instrument Serif (400 italic) |

**Weight contract per font context:**
- **DM Sans (body context):** 400 (regular) + 500 (medium labels) ‚Äî 2 weights
- **Syne (display context):** 700 + 800 ‚Äî 2 weights
- **Instrument Serif:** 400 italic only
- **JetBrains Mono:** 400 only

**Notes on body-role size variations:** These are the same 17px body slot expressed through different font faces or minor size offsets, not separate scale entries:
- Body small (FAQ answers, checklist text, chatbox): 15px ‚Äî 400 DM Sans
- Hero body: 18px ‚Äî 400 DM Sans
- Label / meta (stat labels, fine print): 13px ‚Äî 400‚Äì500 DM Sans
- Label micro (section tags, bar labels, badge text): 11‚Äì12px ‚Äî 700 Syne, uppercase, +0.10‚Äì0.12em tracking
- Mono (URL display, footnote, code): 11‚Äì13px ‚Äî 400 JetBrains Mono

**Removed:** weight 600 is not used anywhere. FAQ question was previously 600 ‚Äî corrected to 700 (Sub-heading slot, Syne).

## Animation and Interaction Contract

### Scroll reveal (`.rv` ‚Üí `.rv.vis`)
- Initial: `opacity: 0`, `transform: translateY(24px)`
- Triggered: `opacity: 1`, `transform: none` ‚Äî easing `cubic-bezier(.16,1,.3,1)` ‚Äî 650ms
- Stagger: `style={{ transitionDelay: delay * 0.1 + 's' }}`
- IntersectionObserver: threshold `0.01`, rootMargin `'20px 0px 0px 0px'`
- Exception: HeroSection inner content renders at full opacity immediately ‚Äî no `.rv` wrapper
- Exception: Elements already fully in viewport on page load should not flash ‚Äî Hero excluded

### AnimNum counter
- Fires once when element enters viewport (threshold `0.5`)
- Duration: 1200ms, ease-out `1 - Math.pow(1 - p, 3)` cubic
- Used in: ProblemSection stat cards for 64 (suffix "%") and 47 (suffix "%")
- "1‚Äì2" stat is NOT animated (non-integer) ‚Äî render static

### F√∂rklararen compare animation (`ExplainerSection`)
- Trigger: IntersectionObserver at 20% threshold
- Delay: 600ms after intersection before `setShowAI(true)`
- Left panel (Google): transitions to `opacity: 0.35`, `filter: blur(1.5px) grayscale(0.6)` ‚Äî 800ms ease
- Right panel (AI): starts `translateX(16px) opacity:0`; transitions to `translateX(0) opacity:1` ‚Äî 700ms `cubic-bezier(.16,1,.3,1)`

### Report card tilt (ExampleSection)
- Default: `transform: rotate(-1deg)` ‚Äî 500ms cubic-bezier(.16,1,.3,1)
- Hover: `transform: rotate(0deg)`

### Reason card lift (ConcreteExampleSection)
- Hover: `transform: translateY(-4px)` ‚Äî 300ms ease

### CTA button hover
- Hero button: `translateY(-2px)` + `box-shadow: 0 8px 24px rgba(232,68,32,0.3)` ‚Äî 200ms
- ScanCTA button: `translateY(-2px)` + `box-shadow: 0 6px 20px rgba(232,68,32,0.4)` ‚Äî 200ms

### Input focus (ScanCTA wrapper)
- Wrapper border transitions: `border-color: #E84420` on `focus-within` ‚Äî 300ms

### Hero pill dot pulse
- `@keyframes pulse { 0%,100% { opacity:1 } 50% { opacity:0.3 } }` ‚Äî 2s infinite

### FAQ accordion (if implemented)
- Open: `max-height: 400px`, `opacity: 1` ‚Äî `max-height .4s cubic-bezier(.16,1,.3,1)`, `opacity .3s`
- Close: `max-height: 0`, `opacity: 0`
- Icon: `+` rotates `rotate(45deg)` when open
