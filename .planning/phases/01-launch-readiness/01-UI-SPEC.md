# Phase 1 — UI Design Contract

> Visual and interaction contract for Phase 1: Landing Page & Scan Flow.
> Generated by gsd-ui-researcher, verified by gsd-ui-checker.
>
> **Primary source:** Handoff `AI Search Scanner v3.html` + `components-v3.jsx` — read in full.
> **Theme override (CONTEXT.md locked):** Background is light (`#FAFAF8`), not dark zinc.
> All other visual decisions — typography, accent color, spacing, animations, card shapes — follow the handoff exactly.

## Design System

| Property | Value |

## Animation and Interaction Contract

### Scroll reveal (`.rv` → `.rv.vis`)
- Initial: `opacity: 0`, `transform: translateY(24px)`
- Triggered: `opacity: 1`, `transform: none` — easing `cubic-bezier(.16,1,.3,1)` — 650ms
- Stagger: `style={{ transitionDelay: delay * 0.1 + 's' }}`
- IntersectionObserver: threshold `0.01`, rootMargin `'20px 0px 0px 0px'`
- Exception: HeroSection inner content renders at full opacity immediately — no `.rv` wrapper
- Exception: Elements already fully in viewport on page load should not flash — Hero excluded

### AnimNum counter
- Fires once when element enters viewport (threshold `0.5`)
- Duration: 1200ms, ease-out `1 - Math.pow(1 - p, 3)` cubic
- Used in: ProblemSection stat cards for 64 (suffix "%") and 47 (suffix "%")
- "1–2" stat is NOT animated (non-integer) — render static

### Förklararen compare animation (`ExplainerSection`)
- Trigger: IntersectionObserver at 20% threshold
- Delay: 600ms after intersection before `setShowAI(true)`
- Left panel (Google): transitions to `opacity: 0.35`, `filter: blur(1.5px) grayscale(0.6)` — 800ms ease
- Right panel (AI): starts `translateX(16px) opacity:0`; transitions to `translateX(0) opacity:1` — 700ms `cubic-bezier(.16,1,.3,1)`

### Report card tilt (ExampleSection)
- Default: `transform: rotate(-1deg)` — 500ms cubic-bezier(.16,1,.3,1)
- Hover: `transform: rotate(0deg)`

### Reason card lift (ConcreteExampleSection)
- Hover: `transform: translateY(-4px)` — 300ms ease

### CTA button hover
- Hero button: `translateY(-2px)` + `box-shadow: 0 8px 24px rgba(232,68,32,0.3)` — 200ms
- ScanCTA button: `translateY(-2px)` + `box-shadow: 0 6px 20px rgba(232,68,32,0.4)` — 200ms

### Input focus (ScanCTA wrapper)
- Wrapper border transitions: `border-color: #E84420` on `focus-within` — 300ms

### Hero pill dot pulse
- `@keyframes pulse { 0%,100% { opacity:1 } 50% { opacity:0.3 } }` — 2s infinite

### FAQ accordion (if implemented)
- Open: `max-height: 400px`, `opacity: 1` — `max-height .4s cubic-bezier(.16,1,.3,1)`, `opacity .3s`
- Close: `max-height: 0`, `opacity: 0`
- Icon: `+` rotates `rotate(45deg)` when open

**Source:** `v3.html` all transition CSS properties extracted verbatim.

## Claude's Discretion Areas

From CONTEXT.md — design decisions left to implementer:

1. **TickerBreak:** Include. Single component, adds visual rhythm. Speed: 40s animation duration.
2. **Förklararen animation:** Follow handoff `SearchChanged()` exactly — IntersectionObserver 20% threshold, 600ms delay.
3. **MistakesSection card style:** No handoff reference. Use: white bg, `border-radius: 20px`, `border: 1px solid #E0DDD8`, `box-shadow: 0 4px 16px rgba(0,0,0,0.04)`. 3-column (desktop), 1-column (mobile ≤700px).

5. **Email capture storage:** `localStorage.setItem('premium_interest_email', email)` — no server write in Phase 1.
